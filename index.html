Privacy Policy

This policy explains how Tab Volume Control handles your data — and the short answer is: it doesn't.

TL;DR
Tab Volume Control collects no data, stores nothing off your device, and never communicates with any server. Everything stays in your browser.
01

Overview

Tab Volume Control is a Chrome browser extension developed by Dollah. It allows users to mute individual tabs and adjust their volume independently, including boosting up to 300% via the Web Audio API.

This privacy policy applies to the Tab Volume Control Chrome extension and describes our practices regarding any information related to the extension's operation.

02

Tab Volume Control does not collect, transmit, store, or share any personal data or user information of any kind.

03

Local Storage

The extension temporarily stores per-tab volume levels in memory during your browser session only. This data exists solely within the Chrome runtime on your device and is cleared automatically when tabs are closed or the browser restarts.

No data is written to disk beyond Chrome's standard extension storage, and nothing is ever sent to an external server.

04

Permissions Explained

The extension requests the following Chrome permissions. Each is used solely to provide core functionality:

Permission Why it's needed
tabs Read and update the mute state of the active tab.
scripting Inject a Web Audio gain node to control tab volume. Only runs when you adjust the slider.
storage Remember the volume level for each tab during the current session.
contextMenus Add a right-click "Mute Tab" shortcut to the toolbar icon.
activeTab Grants temporary access to the current tab only when the user explicitly interacts with the extension. Access is not persistent and requires a direct user gesture.
05

Third Parties

Tab Volume Control does not integrate with any third-party services, analytics platforms, advertising networks, or data brokers. No data is sold, shared, or disclosed to any third party under any circumstances.

06

Remote Code

This extension does not execute any remotely hosted code. All logic runs locally within your browser. The extension is fully compliant with Chrome's Manifest V3 remote code execution restrictions.

07

Children's Privacy

This extension does not target or knowingly collect any information from children under the age of 13, or any age if located in a jurisdiction with a higher minimum age requirement.
